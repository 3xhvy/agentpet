AgentPet

A native macOS menu bar app with a desktop pet that watches your AI coding agents (Claude Code, Codex, Gemini CLI) in real time.

Download for macOS ⭐ Star on GitHub
brew install --cask ntd4996/tap/agentpet

macOS 13+ · Apple Silicon & Intel · MIT licensed

Stop tab-hunting across terminals

Multi-agent monitor

A live menu bar list of every agent: state, what it's doing, and a real-time timer.

At a glance

The menu bar icon shows how many agents run, and turns orange when one needs your input.

Desktop pet

A pet reacts to agent activity — working, waiting, done, celebrate — with optional chat bubbles.

Claude Code · Codex · Gemini

One-tap hook install for precise working / waiting / done detection. Any CLI agent via agentpet run.

Native notifications & sounds

Get alerted (and hear it) when an agent finishes or needs you.

Bring your own pet

Browse an online pet library and download with one click. Map animations to states.
